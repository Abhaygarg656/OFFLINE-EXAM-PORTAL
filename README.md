The **Offline Exam Portal** is a sophisticated, fully functional **Computer Based Test (CBT)** application. It is specifically designed to conduct examinations for Web Designing, featuring a complete workflow from secure student login to automated result calculation and PDF certificate generation.

---

## ğŸš€ Key Features

* **Secure Student Login**: Students can log in using their name and a dynamically generated password pattern (`Name@123`).
* **Dynamic Quiz Engine**: Includes 50 multiple-choice questions (MCQs) covering HTML, CSS, and JavaScript.
* **Live Countdown Timer**: Features a 50-minute auto-timer that automatically submits the exam when time expires.
* **Real-time Progress Tracking**: A visual progress bar helps students track their completion percentage during the test.
* **Automated Certificate Generation**: Upon completion, the system calculates the score and allows students to download a professional PDF certificate.
* **Anti-Cheat Protection**: The portal disables right-clicking and restricts text selection to maintain exam integrity.
